Firebase setup (brief)

1. Create a Firebase project at https://console.firebase.google.com.
2. Enable Authentication (Email/Password + Google Sign-In).
3. Create a Firestore database in production or test mode.
4. Enable Firebase Cloud Messaging (for push notifications).
5. (Optional) Initialize Firebase Functions and Hosting if you plan to deploy admin or API endpoints.

Place your web app config into `frontend/.env` as described in `frontend/README.md`.
